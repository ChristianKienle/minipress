# sao-create-minipress-plugin

> generates a plugin for miniPress

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao create-minipress-plugin my-project
```

### From git

```bash
sao christiankienle/sao-create-minipress-plugin my-project
```
