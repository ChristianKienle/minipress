# Plugin

# Writing a Plugin

A plugin is a normal NPM package. Its entry point (main-file) should export an async function that will be called by miniPress:

module.exports = async ({ options, minipress }) => {
  // plugin implementation goes here
  return {
    // plugin configuration goes here
  }
}

The object passed to your plugin comes with two things:

  1. options: The plugin options-object passed to the plugin by the consumer.
  2. minipress: The miniPress-instance.

You may call methods & get properties on the minipress-instance in case you have special needs. In generat it should be enough to just return a configured plugin object from your function.
